About the Shiny Application

  • This page is dedicated to a Shiny application for analyzing single-cell RNA sequencing (scRNA-seq) datasets.
  • The application is currently under development and will be available here soon. It is expected to be completed within 1-2 weeks.
  • Key features of the app include:
    • Uploading scRNA-seq files for exploratory data analysis (EDA).
    • Preprocessing steps like quality control (QC) filtering and normalization.
    • Cell-type selection, such as partially responsive, progressive disease, malignant, or non-malignant cells.
    • Identification of cell-type-specific highly variable genes (HVGs).
    • Functional analyses and visualizations based on top HVGs.
  • Designed to optimize biomarker discovery, the application is accessible to researchers without programming expertise.
